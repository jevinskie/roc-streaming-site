32 //! Receiver session group.
33 //!
34 //! Contains:
35 //! - a set of related receiver sessions
36 //!
37 //! Session group corresponds to all sessions handled by one receiver slot - a set of
38 //! related complementary endpoints, e.g. one endpoint for audio, one for repair, and one
39 //! for control packets.
40 //!
41 //! Session group creates and removes sessions and routes packets from endpoints to
42 //! sessions with the help of ReceiverSessionRouter.
43 //!
44 //! It also exchanges control information with remote senders using rtcp::Communicator
45 //! and updates routing based on that control information.
